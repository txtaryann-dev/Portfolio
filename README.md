# 🚀 My Personal Portfolio

Welcome to the repository of my personal portfolio website! This site serves as a central hub to showcase my journey as a **Full-Stack Developer**, featuring my projects, technical skills, and professional experience.

🌐 **Live Demo:** [aryann.vercel.app](https://aryann.vercel.app)

----

## 🛠️ Tech Stack

This portfolio is built with modern web technologies to ensure high performance, responsiveness, and a seamless user experience:

* **Frontend:** React.js / Next.js (Tailwind CSS for styling)
* **Animations:** Framer Motion / GSAP
* **Deployment:** Vercel
* **Icons & Assets:** Lucide React / React Icons
